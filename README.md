# 🚗 Гараж — учёт авто

Минималистичное приложение для отслеживания расходов, обслуживания, заправок и напоминаний по твоим машинам. Один файл, работает офлайн, данные хранятся локально в браузере.

![Стек](https://img.shields.io/badge/stack-vanilla%20JS-5b9dff) ![Без сборки](https://img.shields.io/badge/build-none-5b9dff) ![Офлайн](https://img.shields.io/badge/offline-yes-5b9dff)

## Возможности

- **Несколько авто** — добавляй машины (марка, модель, год, пробег, двигатель), переключайся между ними, свой цвет акцента у каждой.
- **5 типов записей:**
  - ⛽ **Заправки** — вводишь литры и цену, расход **л/100 км** и стоимость/км считаются автоматически.
  - 🔧 **ТО / ремонт** — история обслуживания.
  - 💸 **Расходы** — страховка, мойка, штрафы и прочее.
  - ✦ **Улучшения** — «выполнено» или «в планах».
  - 🔔 **Напоминания** — по дате или по пробегу, с подсветкой «Скоро».
- **Обзор** — всего вложено, средний расход топлива, ближайшее ТО, разбивка по категориям.
- **Статистика** — графики трат по месяцам, динамики расхода топлива и структуры расходов (чистый SVG, без библиотек).
- **Экспорт / импорт** данных в JSON — бэкап и перенос между устройствами.
- Тёмная монохромная тема, адаптив под телефон и десктоп.

## Запуск

Просто открой `index.html` в браузере — всё работает офлайн, без сервера и сборки.

Локально через сервер (опционально):

```bash
python3 -m http.server 4178
# затем открой http://localhost:4178
```

## Технологии

Чистый HTML + CSS + JavaScript. Никаких зависимостей и шагов сборки. Данные — в `localStorage`.

## Приватность

Все данные хранятся только в твоём браузере (`localStorage`). Ничего не отправляется на серверы.
